# Left-handed system

![](~/img/LeftHandedSystem2.png "")  


vvvv is using a left-handed coordinate system with:  
* positive X going right  
* positive Y going up   
* positive Z going forward (ie. into the screen).   
This corresponds to the DirectX default.
